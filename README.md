# Tetonor

Multi-player web-based math game.

## Installation

## Dependencies

The depends on:

-   [Nodejs](https://nodejs.org/en/)
-   A browser that [supports](https://caniuse.com/#search=Web%20Components) Web Components.

### Install npm modules

```bash
npm install # Install dependencies
```

### Run the app

```bash
npm start # Run the http server
```
